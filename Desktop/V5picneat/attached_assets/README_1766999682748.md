# Pic N Eat 🚀

AI-powered nutrition tracking app for Purdue dining halls with gamification.

## 🎯 Features

- 📸 **Camera-based meal logging** - Snap photos to track nutrition
- 🎮 **Gamification** - Earn badges and track streaks
- 📊 **Leaderboard** - Compete with friends
- 🔍 **Food search** - Find nutrition info for any food
- 🧮 **Smart calculations** - Personalized calorie goals

## 🛠️ Tech Stack

**Frontend:** React + TypeScript + Vite + Tailwind CSS + Supabase
**Backend:** Python Flask + PyTorch

## 🚀 Quick Start

### Frontend Setup

```bash
cd mobile
npm install
npm run dev
```

### Backend Setup (Optional)

```bash
cd backend
pip install -r requirements.txt
python api.py
```
