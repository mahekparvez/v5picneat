# ⚡ Quick Start - Pic N Eat

## 1️⃣ Clone Repository

```bash
git clone https://github.com/mahekparvez/pic-n-eat-full.git
cd pic-n-eat-full
```

## 2️⃣ Setup Frontend

```bash
cd mobile
npm install
cp .env.example .env
```

Edit `.env` with your Supabase credentials

## 3️⃣ Create Supabase Project

1. Visit https://supabase.com
2. Create free account
3. Create new project
4. Copy Project URL and anon key
5. Paste into `.env`

## 4️⃣ Setup Database

1. Open Supabase Dashboard
2. SQL Editor
3. Copy/paste contents of `database-schema.sql`
4. Click Run

## 5️⃣ Start App

```bash
npm run dev
```

Visit http://localhost:3000 🎉
EOF
